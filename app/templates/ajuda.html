1. Introdução

Conhecendo o sistema

O Sistema Compras Executivo é uma plataforma integrada desenvolvida para facilitar o acompanhamento e gestão de contratos e processos de compras do governo federal.

Objetivos do Sistema
Centralizar informações Contratos e compras em um só lugar
Encontro de contas Facilita compensação entre órgãos
Indicadores KPIs Métricas de gestão estratégica
Otimização de processos Melhoria da eficiência administrativa
Principais Funcionalidades
Análise e Consulta
  • Dashboard com visão geral
  • Consulta avançada de contratos
  • Filtros por UASG
  • Indicadores de performance
Gestão e Relatórios
  • Encontro de contas automatizado
  • Relatórios gerenciais
  • Gestão de usuários
  • Integração com gov.br

2. Acesso ao Sistema

Como fazer login no sistema

Autenticação via gov.br

O sistema utiliza a autenticação única do gov.br para garantir segurança e praticidade.

Pré-requisito: É necessário ter uma conta ativa no portal gov.br para acessar o sistema.
Passo a Passo para Login
1
Acesse a URL do sistema Digite o endereço no seu navegador
2
Clique em "Entrar com gov.br" Botão azul na página de login
3
Insira suas credenciais CPF e senha do portal gov.br
4
Complete a autenticação Siga os passos do multifator, se solicitado
5
Aguarde o redirecionamento Você será levado ao dashboard principal
Problemas de Acesso
Verifique se seu CPF está cadastrado no sistema
Confirme se possui as permissões necessárias
Entre em contato com o suporte técnico

5. Encontro de Contas

Compensação financeira entre órgãos

O módulo de Encontro de Contas permite a compensação financeira entre órgãos federais, identificando débitos e créditos recíprocos.

Conceitos Importantes
Débito Valor que um órgão deve a outro
Crédito Valor que um órgão tem a receber
Compensação Cancelamento mútuo de débitos e créditos
Saldo Diferença após compensação
Importante: O processo de encontro de contas deve ser executado com cuidado, pois afeta diretamente a situação financeira dos órgãos envolvidos.

6. Filtros UASG

Configuração de unidades administrativas

As UASGs (Unidades Administrativas de Serviços Gerais) são fundamentais para a organização dos dados no sistema.

O que é UASG

UASG é o código que identifica unicamente cada unidade administrativa responsável por atividades de administração geral no âmbito do SISG (Sistema de Serviços Gerais).

As configurações de UASG determinam quais dados você pode visualizar no sistema. Entre em contato com o administrador para alterar suas permissões.

10. FAQ

Perguntas frequentes

Como recuperar minha senha?

Para recuperar sua senha, acesse o portal gov.br e utilize a opção "Esqueci minha senha".

Por que não consigo ver todos os contratos?

Sua visualização está limitada às UASGs configuradas para seu usuário. Entre em contato com o administrador.

Como exportar relatórios?

Use os botões de exportação disponíveis em cada módulo para baixar dados em Excel ou PDF.
